# AI Agent Guidelines

This repository is a personal development workspace. AI coding assistants may help with normal software engineering tasks, including reading files, running commands, editing code, debugging, testing, and explaining changes.

## Working Style

* Prefer small, focused changes.
* Preserve unrelated user changes in the worktree.
* Explain important assumptions and tradeoffs.
* Run relevant checks when making code changes.
* Ask for clarification when a request is ambiguous or risky.
